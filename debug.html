🔧 常見問題解決方案

如果 Supabase 連接失敗：

  • 檢查 SUPABASE_URL 和 SUPABASE_ANON_KEY 是否正確
  • 確認 Supabase 項目的 images bucket 已創建且設為公開
  • 檢查 GitHub Pages 域名是否在 Supabase 的允許域名列表中

如果語音功能不工作：

  • ResponsiveVoice 在 GitHub Pages 上可能載入失敗
  • 應用會自動降級到瀏覽器內建語音
  • 確保瀏覽器支持 Web Speech API

如果圖片無法載入：

  • 檢查 HTTPS 混合內容問題
  • 確認 Supabase Storage 的 CORS 設置正確
  • 檢查圖片 URL 是否可以直接訪問
